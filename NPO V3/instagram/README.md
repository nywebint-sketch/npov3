# Данные с Instagram для НПО Мелодия

Профиль: **https://www.instagram.com/npo_melody/**

Instagram не отдаёт данные при автоматическом запросе (403), поэтому информацию нужно подставить вручную после просмотра профиля.

## Что можно взять с профиля и подставить на сайт

1. **Ссылка на профиль** — уже прописана в блоке «Контакты» на сайте.
2. **Описание (bio)** — скопируй текст из шапки профиля и при необходимости вставь в описание лейбла/раздела (например, в подзаголовок или meta description).
3. **Ссылка из профиля** — если в bio указан сайт или другая ссылка, её можно добавить в контакты или в данные ниже.

## Файл данных

В `instagram-data.json` можно хранить скопированные с Инсты данные (bio, ссылка из профиля и т.д.) для использования в вёрстке или скриптах.
